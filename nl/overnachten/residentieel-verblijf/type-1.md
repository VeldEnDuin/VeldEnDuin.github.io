# TYPE 1 - CARAVAN 

Deze plaatsen zijn iets kleiner qua oppervlakte maar bieden evengoed een instant vakantiegevoel. 


## Goed om weten

- teller voor elektriciteit - forfait water
- gasflessen
- geen wintervaste wateraansluiting - sanitairgebouw blijft open
